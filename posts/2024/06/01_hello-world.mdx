## Next.jsの基本

Next.jsは、Reactベースのフレームワークで、サーバーサイドレンダリング（SSR）や静的サイト生成（SSG）などの機能を提供します。

```jsx
// サンプルコード
function HomePage() {
  return (
    <div>
      <h1>Hello, Next.js!</h1>
    </div>
  )
}

export default HomePage
```
